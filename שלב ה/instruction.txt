הוראות התקנה והפעלה - מערכת ניהול תוכן ומדיה
מסמך זה מפרט את השלבים הנדרשים להתקנה, הגדרה ושימוש בממשק הגרפי לניהול בסיס הנתונים.

1. דרישות קדם
לפני הפעלת המערכת, יש לוודא שהרכיבים הבאים מותקנים ומוכנים:

Python 3: יש לוודא שמותקנת על המחשב גרסה עדכנית של Python.

ספריית psycopg2: זוהי הספרייה המאפשרת ל-Python להתחבר לבסיס הנתונים של PostgreSQL. ניתן להתקין אותה באמצעות הפקודה הבאה בשורת הפקודה (CMD/Terminal):

Bash

pip install psycopg2-binary
שרת PostgreSQL: יש לוודא ששרת בסיסי נתונים של PostgreSQL מותקן, פועל ונגיש מהמחשב בו תרוץ התוכנה.

בסיס נתונים קיים: הקוד מיועד להתחבר לבסיס נתונים קיים. יש לוודא שבסיס הנתונים והטבלאות הרלוונטיות (title, franchise, belongs_to, וכו') כבר קיימים על השרת.

2. הגדרת חיבור לבסיס הנתונים
כדי שהמערכת תוכל להתחבר לבסיס הנתונים שלך, יש לערוך את קובץ ה-Python ולעדכן את פרטי החיבור.

פתח את קובץ ה-Python (.py) בעורך טקסט או סביבת פיתוח.

נווט לפונקציה connect_to_db (בסביבות שורה 20).

שנה את ערכי הפרמטרים הבאים בהתאם להגדרות השרת שלך:

dbname: שם בסיס הנתונים שאליו תתחבר (כרגע: "mydatabase").

user: שם המשתמש שלך ל-PostgreSQL (כרגע: "mynaoruser").

password: הסיסמה של המשתמש (כרגע: "Yukhnxyukh1!").

host: כתובת השרת (כרגע: "localhost", מתאים לשרת שרץ על אותו מחשב).

port: הפורט של השרת (כרגע: "5432", ברירת המחדל של PostgreSQL).

דוגמה לקוד לאחר עריכה:

Python

def connect_to_db(self):
    """חיבור לבסיס הנתונים PostgreSQL"""
    try:
        self.conn = psycopg2.connect(
            dbname="your_db_name",      #  <-- שנה כאן
            user="your_username",      #  <-- שנה כאן
            password="your_password",  #  <-- שנה כאן
            host="your_host_address",  #  <-- שנה כאן (לרוב 'localhost')
            port="5432"
        )
        messagebox.showinfo("הצלחה", "התחברנו בהצלחה לבסיס הנתונים!")
    except Exception as e:
        messagebox.showerror("שגיאה", f"שגיאה בחיבור לבסיס הנתונים:\n{str(e)}")
        sys.exit()
3. הפעלת המערכת
לאחר עדכון פרטי החיבור, ניתן להפעיל את המערכת.

פתח את שורת הפקודה (CMD/Terminal).

נווט לתיקייה בה שמור קובץ ה-Python.

הרץ את הפקודה הבאה:

Bash

python your_script_name.py
(החלף את your_script_name.py בשם הקובץ שלך).

עם ההפעלה, תופיע הודעה המאשרת שהחיבור לבסיס הנתונים הצליח. לאחר מכן, יוצג התפריט הראשי של המערכת.

4. שימוש במערכת
המערכת מחולקת למספר מסכי ניהול, הנגישים מהתפריט הראשי.

תפריט ראשי
המסך הראשי מכיל ארבעה כפתורי ניווט למסכי הניהול השונים וכפתור יציאה.

ניהול כותרים (Titles): מסך לניהול רשומות בטבלת title.

ניהול פרנצ'ייזים (Franchises): מסך לניהול רשומות בטבלת franchise.

ניהול קישורים (Belongs To): מסך לניהול הקשרים בין כותרים ופרנצ'ייזים בטבלת belongs_to.

שאילתות ופונקציות מתקדמות: מסך להרצת שאילתות מורכבות ופונקציות שהוגדרו מראש בבסיס הנתונים.

יציאה: סוגר את החיבור לבסיס הנתונים ויוצא מהתוכנה.

מסכי ניהול (כותרים ופרנצ'ייזים)
מסכי ניהול הכותרים והפרנצ'ייזים פועלים באופן דומה. כל מסך מחולק לשני חלקים עיקריים:

טופס הוספה/עדכון: בחלק העליון, טופס עם שדות למילוי.

רשימת נתונים: בחלק התחתון, טבלה המציגה את כל הרשומות הקיימות מהטבלה הרלוונטית.

פעולות אפשריות:

הוספה (Add):

מלא את כל השדות בטופס (למעט שדות אופציונליים).

לחץ על כפתור "הוסף". הרשומה החדשה תתווסף לבסיס הנתונים והטבלה תתרענן.

עדכון (Update):

לחץ על הרשומה שברצונך לעדכן בטבלת הנתונים. פרטיה ימולאו אוטומטית בטופס.

שנה את הערכים הרצויים בשדות הטופס. שים לב: לא ניתן לשנות את שדה המזהה (title_id / franchise_id).

לחץ על כפתור "עדכן".

מחיקה (Delete):

בחר את הרשומה למחיקה מהטבלה.

לחץ על כפתור "מחק". תופיע הודעת אישור.

ניקוי (Clear):

לחץ על כפתור "נקה" כדי לרוקן את כל השדות בטופס.

מסך ניהול קישורים (Belongs To)
מסך זה מאפשר לקשר בין רשומת title קיימת לרשומת franchise קיימת.

הוספת קישור:

הזן את מזהה הפרנצ'ייז ואת מזהה הכותר בשדות המתאימים.

לחץ על "הוסף קישור".

מחיקת קישור:

בחר את הקישור הרצוי מהטבלה. פרטי המזהים יופיעו בשדות הטופס.

לחץ על "מחק קישור".

מסך שאילתות ופונקציות מתקדמות
מסך זה מחולק לשתי לשוניות:

לשונית "שאילתות מורכבות":

מכילה מספר כפתורים, כאשר כל כפתור מייצג שאילתה מוגדרת מראש.

לחץ על אחד הכפתורים כדי להריץ את השאילתה.

התוצאות יוצגו בטבלה שבתחתית הלשונית.

לשונית "פונקציות ופרוצדורות":

סטטיסטיקות יוצר:

הזן מזהה יוצר (מספר) בשדה הקלט.

לחץ על "הרץ פונקציה".

התוצאות (סטטיסטיקות על היוצר) יוצגו בתיבת הטקסט התחתונה.

עדכון מספר כותרים בפרנצ'ייז:

הזן מזהה פרנצ'ייז (מספר) בשדה הקלט.

לחץ על "הרץ פרוצדורה".

הפרוצדורה תעדכן את ספירת הכותרים בטבלת franchise. הודעת אישור והתוצאה המעודכנת יוצגו בתיבת הטקסט.

בכל מסך קיים כפתור "חזרה לתפריט הראשי" המאפשר לחזור למסך הבית.
